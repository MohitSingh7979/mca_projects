This been a simple Webpage demonstrating some aspects of HTML.

Here are some of tags used in html

HTML
The starting (root) tag enclosing whole html document.
BODY
The actual tag which enclose everything to shown on viewrs screen.
HEAD
The tag to write metadata about page, linking things etc.
TITLE
The tag used to given a title to page which is usually seen in tab of browser.
P
The tag used to create a paragraph.
DL
The tag used to create a Data List.
DT
The tag used to create term to be defined.
DD
The tag used to create definition of term defined with `dd` tag.
